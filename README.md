## BKC Laravel 9 | Twill CMS package

#### Environment Requirements
- php >=8.1
- mysql >=8.0
- npm / node.js

#### Installation

- Clone repository
- Composer install
- php artisan key:generate
- php artisan migrate
- npm install
- (dev) npm run dev . watch changes in files an sync on localhost
- (prod) npm run build
